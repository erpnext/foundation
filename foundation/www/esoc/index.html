ERPNext Summer of Code

ERPNext Summer of Code is an initiative by the ERPNext Foundation, in similar lines to Google Summer of Code (GSoC) to help bring more young and budding developers into the world of Open Source Software development. Students will spend 3 months of their summer time working with ERPNext and contributing to open source development, all while earning a stipend!
